# 21.3.2 Standards And Bodies

_Page Type: Comparison Page | Maturity: Draft_

This file keeps formal standards, public bodies, and reference frameworks in one place so they can be compared without overwhelming the chapter front door.

## How To Use This File

- Frameworks solve different jobs and should not be treated as interchangeable.
- A standard is only useful when translated into design, control, or operating decisions.
- Public bodies and standards should not be scored like software platforms.
